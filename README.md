# КВИК - Управление проектами

КВИК - это веб-приложение для управления проектами и задачами. Позволяет создавать проекты, ставить задачи, отслеживать статус выполнения и анализировать прогресс команды.

## Функционал

1. Авторизация и регистрация пользователей
2. Создание, редактирование и удаление проектов
3. Выбор цвета фона для каждого проекта
4. Управление задачами (CRUD) с приоритетами (Высокий, Средний, Низкий) и дедлайнами
5. Канбан-доска с тремя статусами: «Нужно сделать», «В процессе», «Готово»
6. Поиск, фильтрация и сортировка задач
7. Аналитика: круговая диаграмма, прогресс проектов, рейтинг участников
8. Настройки: светлая/тёмная тема, управление командой (приглашение участников)
9. Адаптивный дизайн
10. Docker-контейнеризация для простого развёртывания

## Технологии

HTML5 - Структура страниц
CSS3 - Стилизация, адаптив, тёмная/светлая тема 
JavaScript - Логика приложения, фильтрация, сортировка
Chart.js - Построение графиков в аналитике
Docker - Контейнеризация приложения

## Запуск через Docker

### Предварительные требования

     Установленный [Docker Desktop](https://www.docker.com/products/docker-desktop/)

### Шаги для запуска

# 1. Клонировать репозиторий
     git clone https://github.com/t11katya/KVIK-project.git
     cd KVIK-project

# 2. Собрать Docker-образ
    docker build -t kvik-app .

# 3. Запустить контейнер
    docker run -d -p 8080:80 --name kvik-container kvik-app
